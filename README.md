# Android WebView (Ghost)

An Android application that wraps a web page in a native WebView. The app is named "Ghost" internally and is built with Kotlin using the modern Gradle Kotlin DSL.

## Tech Stack

- Language: Kotlin
- Build: Gradle (Kotlin DSL `.gradle.kts`)
- Min SDK: configurable via `app/build.gradle.kts`

## Requirements

- Android Studio
- Java 11+
- Android SDK

## Build & Run

```bash
./gradlew assembleDebug
./gradlew installDebug
```
